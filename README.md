# truemail

Быстрый, красивый, кроссплатформенный опенсорс почтовый клиент на Rust.
Нативный десктоп (Windows, macOS, Linux) плюс локальный API для LLM-интеграций.

Всё на открытых стандартах: IMAP/SMTP/MIME, iCalendar, vCard, CalDAV/CardDAV -
проприетарные протоколы (Exchange EWS, Graph, JMAP) приводятся к RFC. Приватность
и скорость в основе: локальное хранение почты, секреты в системном keychain,
мгновенная доставка писем, встроенный AI (суммаризация, черновики, умный поиск)
с приоритетом локальных моделей.

Проект в стадии проектирования.
