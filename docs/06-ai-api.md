# Локальный REST и MCP API

API запускается только на `127.0.0.1`; стандартный порт — `34981`. Управление
сервером, клиентами и журналом находится в настройках truemail.

Клиент получает случайный bearer-токен один раз. Открытый токен хранится только
в системном keychain, в SQLCipher-базе остаётся SHA-256 для проверки. Права
выдаются отдельно: чтение, поиск, отправка, метки, календарь и сеть. Каждый
разрешённый или отклонённый вызов записывается в `api_audit`.

- `GET /health` — состояние сервера.
- `GET /v1/tools` — доступные клиенту инструменты.
- `POST /v1/tools/{name}` — REST-вызов инструмента.
- `POST /mcp` — MCP JSON-RPC (`initialize`, `tools/list`, `tools/call`).

REST и MCP используют один исполнитель инструментов и одну проверку capability,
поэтому способы доступа не расходятся по поведению и аудиту.
