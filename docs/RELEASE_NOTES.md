Первый крупный выпуск после 0.1.0. Что изменилось:

## Почта и провайдеры
- Инкрементальная синхронизация Gmail через History API вместо полной перезагрузки; клиентский ограничитель частоты запросов, чтобы не упираться в лимиты Gmail (ошибки 429)
- Подключение корпоративного Microsoft Exchange (Autodiscover + EWS + NTLM/Negotiate), в том числе self-hosted
- Подключение Outlook через OAuth
- Подключение произвольных IMAP-аккаунтов по паролю приложения
- Транспорт JMAP
- Экспорт письма в файл .eml

## Календарь и контакты
- Синхронизация календарей и контактов Exchange (EWS)
- Инкрементальная синхронизация вспомогательных данных (CardDAV sync-токены, ctag/syncToken) - не перезабирает весь набор при каждом обновлении
- Участники и напоминания событий

## Функции
- Отложить письмо (snooze), подписи, шаблоны
- Правила обработки почты выполняются в ядре и работают без открытого окна
- Умные папки перенесены в ядро
- Локальный REST и MCP API
- Экспорт и восстановление ключей шифрования (резервная копия хранилища)
- Подписанные автообновления приложения

## Интерфейс и производительность
- Виртуализация списка писем, drag-and-drop и свайпы между папками
- Кэш разобранного MIME - письма открываются быстрее
- Модульная декомпозиция интерфейса, улучшенный онбординг и настройки
- Аватары контактов одинакового размера
- Централизованный лог ошибок

## Платформы
- Сборки для Windows и macOS (Apple Silicon) в дополнение к Linux
- Хранилище на SQLCipher 4.17 (шифрование локальной базы)
