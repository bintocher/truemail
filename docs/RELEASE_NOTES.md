Выпуск про календарь: уведомления об изменениях встреч, ответ на приглашение
одним кликом и полное управление событиями и контактами во всех провайдерах.

## Календарь
- Уведомления об изменениях встреч: новая, перенесена, отменена, переименована, сменилось место, изменился состав участников
- В карточке уведомления дата, время, место, организатор и число участников
- Ответ на приглашение прямо из уведомления - "Пойду", "Возможно", "Не пойду"; организатору уходит ответ по iTIP, решение можно изменить позже
- При лавине изменений вместо потока карточек одна сводная
- Об уже прошедших встречах уведомления не приходят
- Выбранный вид (день, неделя, месяц) сохраняется до следующего запуска
- Сетка растягивается на всю высоту области и учитывает заданные рабочие часы

## Почта
- Уведомление приходит только о письмах, которые действительно пришли при работающей программе; повторные карточки об одном письме больше не появляются
- Создание папки на сервере из контекстного меню - для IMAP, Exchange, JMAP и Gmail
- Пометка "важное" больше не сбрасывает признак прочитанности при синхронизации

## Провайдеры
- Exchange: создание, изменение и удаление событий и контактов (раньше только чтение)
- CalDAV и CardDAV для iCloud, Mail.ru, Outlook и произвольных серверов, с автоопределением адресов календарей и контактов

## Прочее
- Отправка письма фоном: окно закрывается сразу, о результате сообщает уведомление
- Проверка обновлений каждые 6 часов, а не только при запуске

## Безопасность
- Сохранение вложений не может записать файл за пределы выбранной папки, даже если имя вложения подобрано специально
- Отписка в один клик не обращается к адресам внутренней сети и не идёт по переадресациям
- В письмах отображаются только вложенные картинки; прочее встроенное содержимое блокируется
- Адреса почты в журналах маскируются, журналы хранятся 7 дней и в обычной сборке пишутся короче
