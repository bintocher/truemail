# Иконки приложения

Готовый комплект иконок для Windows, macOS, Linux, iOS и Android. Источник
истины находится в `assets/brand/truemail-app-icon.svg`.

Перегенерировать весь комплект из корня репозитория:

```sh
cargo tauri icon assets/brand/truemail-app-icon.svg -o apps/desktop/src-tauri/icons
```
